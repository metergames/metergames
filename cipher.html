Gameplay

  • Daily Challenge: One fresh puzzle every day with increasing difficulty.
  • Multiple Cipher Types: Binary, Caesar, Hex, XOR, substitution, and more.
  • Streaks & Archive: Track consecutive solves and unlock past puzzles.
  • Hints: Optional hints to help you progress without spoiling the solution.

Platform & UX

  • Platform: Android (targeting modern devices).
  • Design: Clean, high-contrast UI with large, readable monospaced output for ciphers.
  • Offline Support: Play the latest unlocked puzzles without network connectivity.
  • Accessibility: Adjustable text size and high-contrast theme support.

Development Notes

Built with a lightweight local persistence layer for streaks and archive. Planned features include a puzzle editor, creator mode, and optional cloud sync.
