## 🚀 About Me

I am a versatile software developer with a strong background in full-stack web development, AI model optimization, and game architecture. I am currently pursuing a Computer Science DEC and actively developing independent projects under Meter Games. Recently, I have been building "Pit Wall," a real-time Formula 1 data dashboard using React and TypeScript, alongside "Parker," a comprehensive Unity PC simulation. Whether I am evaluating complex AI-generated software or architecting scalable back-end infrastructure, I am passionate about writing clean, secure, and maintainable code.
